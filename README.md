# 💳 CreditWise: Loan Approval Prediction & Risk Analysis

Built an end-to-end supervised ML pipeline using KNN, Logistic Regression and Naive Bayes to predict loan approval.  Implemented Binary classification along with EDA, feature engineering &amp; model evaluation (Precision, Recall, F1).
---

## 🚀 Project Overview

Financial institutions receive thousands of loan applications every day. Evaluating each application manually can be time-consuming and prone to errors.

This project leverages Machine Learning techniques to automate the loan approval process by analyzing applicant profiles and predicting whether a loan should be approved.

---

## 🎯 Objectives

- Predict loan approval status accurately.
- Identify the most influential factors affecting loan decisions.
- Perform data cleaning and feature engineering.
- Compare machine learning algorithms.
- Evaluate models using classification metrics.

---

## 📊 Dataset Features

| Feature | Description |
|----------|-------------|
| Applicant Income | Monthly income of applicant |
| Coapplicant Income | Monthly income of co-applicant |
| Loan Amount | Requested loan amount |
| Loan Amount Term | Repayment duration |
| Credit Score | Applicant creditworthiness |
| DTI Ratio | Debt-to-Income Ratio |
| Employment Status | Employment category |
| Education | Educational qualification |
| Property Area | Urban / Semi-Urban / Rural |
| Loan Approved | Target variable |

✅ Credit Score is one of the strongest predictors of loan approval.

✅ Applicants with lower Debt-to-Income (DTI) ratios are more likely to receive approval.

✅ Feature engineering improved model performance.

✅ Logistic Regression achieved strong classification performance.
